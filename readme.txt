A simple WordPress plugin that displays the date that the plugin was last updated on. 

Video Tutorial

[youtube https://www.youtube.com/watch?v=U4wNjKMAv48]

== Installation ==


1. Go to the Plugins> Add New> Upload in you admin dashboard.

2. Upload the zip file and install.
3. Go to the Plugins section in your admin dashboard and activate.
